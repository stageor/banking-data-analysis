# Banking Data Analysis Project

## Objective
Perform exploratory data analysis on a synthetic banking dataset and generate automated visual reports in PDF format.

## Dataset
Mock banking data with the following fields:
- Customer ID
- Age
- Gender
- Account balance
- Loan amount
- Account activity status
- Transaction count

## Analysis Performed
- Active vs inactive account distribution
- Average transactions by age
- Account balance vs loan amount
- Loan ownership distribution
- Account status distribution

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Outputs
- Automated charts saved as PNG files
- Consolidated PDF report generated programmatically

## Files
- `Banking_Report.ipynb` – Complete analysis notebook
- `banking_report.pdf` – Final report
- `*.png` – Generated visualizations
